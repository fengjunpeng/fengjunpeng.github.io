## 斐波那契数列

> 大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。  
n <= 39

## 思路

- 斐波那契数列数列大家都知道，前两项的元素数值都是固定的，从第三项开始，元素的数值等于前两项元素的数值之和，采用递归方式解决问题。

## java代码

    public class Solution {
        public int Fibonacci(int n) {
            if(n == 0){
                return 0;
            }
            if(n == 1){
                return 1;
            }
            return Fibonacci(n - 2) + Fibonacci(n - 1);
        }
    }
